    Class WGFX

    Main WGFX class for handling WebGPU graphics effects.

    處理 WebGPU 圖形特效的主要控制器。

    Example 

    const wgfx = await WGFX.create({ device, effectCode, width: 1280, height: 720 });
    const output = await wgfx.process(videoElement);
    // 渲染完成後關閉
    wgfx.dispose();
