    Class WGFXRuntime

    Core runtime for WGFX, managing resources, pipelines, and execution.

    WGFX 的核心運行時環境，負責協調資源管理、管線編譯與指令調度執行。 它是連接高層 API 與底層 WebGPU 操作的橋樑。
